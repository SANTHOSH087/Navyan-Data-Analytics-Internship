# 📊 Navyan Data Analytics Internship

This repository contains the projects completed as part of the **Navyan Data Analytics Internship**. The projects demonstrate skills in data analysis, customer segmentation, and real-time dashboard development using Python and Streamlit.

## 🚀 Projects

### 📈 Project 1: Sales Data Analysis Dashboard
**Description:**
- Analyzed sales data using Python.
- Performed data cleaning and preprocessing.
- Generated insights using data visualization.
- Identified sales trends and key performance metrics.

**Technologies Used:**
- Python
- Pandas
- Matplotlib

---

### 👥 Project 2: Customer Segmentation Analysis
**Description:**
- Segmented customers based on purchasing behavior.
- Applied clustering techniques for customer analysis.
- Visualized customer groups for business insights.

**Technologies Used:**
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

### 🌍 Project 3: Real-Time Data Analytics Dashboard
**Description:**
A Streamlit dashboard that displays live data from multiple online sources.

**Features:**
- 🌤️ Live Weather Updates
- 📈 Apple Stock Price
- ₿ Bitcoin Price
- 🔄 Real-time API Integration

**Technologies Used:**
- Streamlit
- Python
- Pandas
- Requests
